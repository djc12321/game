# 半夜偷玩手机 — 重构设计文档

## 一、核心理念

一款 **元游戏**：你扮演深夜偷偷玩手机的孩子，在被妈妈发现之前，控制手机里的潜行小游戏收集所有光球。
**双层游戏循环**：3D 卧室（被发现的恐惧）+ 2D 手机（潜行的乐趣）。

## 二、与原文档的关键改进

| 原设计 | 重构改进 | 理由 |
|---|---|---|
| 头部在枕头看手机，看不到自己身体 | **第一人称双手双臂建模**，举手机姿态 | 沉浸感与真实感 |
| 妈妈：球+圆柱+圆锥粗略模型 | **分段身体（头/发髻/裙/上下肢）+ 走路动画** | 精良建模，腿摆动+手臂摆动+回身看你 |
| 2D 游戏：圆形检测 360° 视野 | **机器人视野扇形** + 障碍遮挡视线 | 真正的潜行机制（背后/掩体安全） |
| 仅触摸操作 | **统一输入抽象**：键盘 WASD + 鼠标拖看 + 触摸双区操作 | 双端适配 |
| 资源依赖 11+ 张图、7 个音频 | **零外部资源**：全部程序化几何 + WebAudio 合成音效 | 直接运行，秒部署 |
| 仅"放下手机"单一反应 | **冲刺、连击、视野阻挡、举/放手机**多机制 | 玩法深度 |
| 手机布局：屏占满 | 顶部状态栏 + 底部冲刺/分数/连击栏 | 信息分层 |

## 四、3D 视觉系统

### 第一人称双手
- 每只手 = 上臂袖子（圆柱，睡衣色）+ 前臂（圆柱，肤色）+ 手掌（Box）+ 拇指 + 4 根手指（圆柱）
- 附加在 `camera` 上，跟随视角；位置 (±0.32, -0.5, -0.55) 等，斜向中央上方延伸
- 手机居中夹在双手之间

### 手机
- BoxGeometry 16×30×1.2 cm 黑边框 + Plane 屏幕（CanvasTexture）
- **举起 ↔ 放下** 平滑插值 4Hz：屏幕透明度同步降为 0；放下时手机翻转到床侧不遮挡视线

### 妈妈精良模型
- **头**：球体（肤色 0xe8c8a8） + 半球头发 + 球体发髻
- **眼**：两个红色 BasicMaterial 小球（暗中发光感）
- **躯干**：圆台 + 下接圆锥裙摆（深紫 0x3a1a3a）
- **手臂**：上臂(袖) + 前臂(肤) 组成 Group，**rotation.x 同步走路相位摆动**
- **腿**：圆柱 + Box 鞋，同样按相位摆动；`Math.sin(walkPhase) * 0.6`
- **行为**：从门外 → 探头 → 走到床边检查 → 转身离开；`atan2(dx, dz)` 实时朝向移动方向

### 8 路灯光（与原文档一致，强化脉冲算法）
环境 / 月光（投影） / 手机 / 走廊 / 红色预警 / 妈妈跟随 / 闪电 / 台灯

### 动态天气
- 600 个雨点 BufferGeometry+Points
- 闪电：每帧 0.0008 概率触发，强度 4-8 持续 100-220ms

## 五、2D 潜行游戏

### 玩家
- 绿色发光小球，物理：加速度 2200 / 最大速度 480 / 摩擦 0.0015^dt
- **冲刺**：1100 速度 0.18s，冷却 1.5s（电脑 Shift / 手机按钮）

### 光球
- 18px 半径，呼吸动画 sin(phase*4)
- 收集时：14 颗金色粒子爆散 + 触发 `audio.collect/combo`
- **连击**：2.5s 内连续收集 +1，得分 = 100 + combo*25

### 机器人
- 3 点路径巡逻 + 平滑转向
- **视野扇形**：260px 距离，半角 0.55 rad（约 63°）
- **遮挡检测**：line-rect 采样 12 点；任何障碍挡线则视线无效
- 警觉积累 0.35s 才被判定抓到，给玩家短暂应急
- 直接接触 = 立即被抓

### 障碍
- 床(180×240) / 衣柜(140×90) / 桌(200×110)
- 圆-矩形最近点法的贴墙滑动碰撞

### 手电筒效果
- 玩家周围 80→360px 径向遮罩（multiply 混合）
- 远处变暗，自然限制可视范围

## 六、妈妈状态机（GameManager）

```
IDLE ─── (idleTimer<=0) ──► APPROACHING (1.2s 走廊光淡入)
APPROACHING ──► AT_DOOR (0.6s, 启动反应窗口 reactionWindow)
AT_DOOR ──► PEEKING (1.0s, 妈妈头出现在门缝)
  ├── 玩家放下手机 → 等待妈妈结束阶段
  └── reactionWindow 用尽且手机未放 → 立即输
PEEKING ──► CHECKING (3.5s, 妈妈走到床边)
  └── 检查中举手机 → 立即输
CHECKING ──► LEAVING (2.0s, 妈妈走回门口)
LEAVING ──► IDLE (按难度调度下次出现)
```

## 七、双端控制

### 桌面 Web
| 操作 | 键 |
|---|---|
| 2D 小球移动 | W/A/S/D 或方向键 |
| 冲刺 | Shift |
| 看房间 | 鼠标按下拖拽（左/右键） |
| 放下/拿起手机 | 空格 |
| 暂停 | Esc |
| 大按钮 | 警告时点击屏幕中央按钮 |

### 移动 Web
| 操作 | 手势 |
|---|---|
| 2D 小球移动 | **左半屏触摸拖拽**（虚拟摇杆出现在按下位置） |
| 看房间 | **右半屏触摸拖拽** |
| 冲刺 | 右下角 **冲刺按钮** |
| 放下/拿起手机 | 屏幕中央 **大按钮**（妈妈预警时自动出现） |

## 八、难度与关卡

10 关递进 + 5 档难度（速度 / 妈妈频率 / 时限 / 光球数 / 反应窗口）。详见 `constants.ts`。

## 九、技术亮点

1. **零外部资产**：所有几何为 Three.js primitives 程序化拼装，所有音效为 WebAudio 合成
2. **CanvasTexture 双层**：2D 游戏每帧绘制到 Canvas，作为 3D 手机屏幕纹理
3. **统一输入层**：`InputManager` 屏蔽端差异，外部只需 `poll()` 取标准化 axis
4. **响应式 viewport**：safe-area-inset 支持 iPhone 刘海；touch-action: none 防误滚动
5. **存档**：localStorage `midnight-phone-save`，记录最高关卡、难度、音效、各关卡最高分
