# 半夜偷玩手机 🌙📱

深夜潜行收集 + 装睡躲妈妈 — 一款 React + Three.js 的双层元游戏。

- 🎨 [设计文档 DESIGN.md](DESIGN.md) — 架构、改进点、技术细节
- 🎮 [游戏教程 TUTORIAL.md](TUTORIAL.md) — 操作、技巧、运行方式

## 快速开始

```bash
npm install
npm run dev
```

打开 http://localhost:5173 即可。手机访问用同一 Wi-Fi 的电脑 IP。

## 适配

- **桌面**：WASD 移动 + 鼠标拖拽看房间 + 空格放手机 + Shift 冲刺
- **移动**：左半屏摇杆 + 右半屏视角 + 冲刺/放手机按钮

## 特点

- ✅ **零外部资源**：所有 3D 几何与音效全部程序化生成
- ✅ **第一人称双手双臂建模** — 真正能看到自己举着手机
- ✅ **精良妈妈模型** — 分段身体 + 走路动画 + 朝向你
- ✅ **真潜行机制** — 视野扇形 + 障碍遮挡 + 冲刺保命
- ✅ **桌面/移动响应式 UI**
