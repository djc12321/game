# 🎮 半夜偷玩手机 — 游戏教学

> 一个夜深人静的潜行小游戏。妈妈随时可能推门而入，你能撑过 10 关吗？

## 🌙 故事背景

凌晨 1 点，你蜷在被窝里，借着月光偷偷打开手机玩潜行游戏。
但走廊那头… 妈妈起夜了。

## 🎯 你的目标

| 双层目标 | 描述 |
|---|---|
| **2D 内层**：绿色小球 | 收集屏幕里所有金色光球，避开巡逻机器人 |
| **3D 外层**：你自己 | 妈妈来查房时，**立刻放下手机装睡**，绝不能被抓到 |

## 🕹 操作方式

### 💻 电脑 Web 端

| 按键 | 动作 |
|------|------|
| `W` `A` `S` `D` / `方向键` | 控制 2D 小球移动 |
| `Shift` | 冲刺（短距高速突进，1.5 秒冷却） |
| **鼠标按住拖拽** | 环顾 3D 卧室（看门、窗、家具） |
| `空格` | 放下手机 / 拿起手机 |
| `Esc` | 暂停游戏 |

### 📱 手机 Web 端

| 手势 | 动作 |
|------|------|
| **左半屏拖拽** | 虚拟摇杆控制小球移动 |
| **右半屏拖拽** | 环顾 3D 房间 |
| 右下 **「冲刺」** 按钮 | 冲刺 |
| 屏幕中央 **大按钮** | 放下 / 拿起手机（按需出现） |
| 顶部 ⏸ / ✕ 按钮 | 暂停 / 退出 |

> 移动端建议横屏或竖屏均可，UI 自适应。

## ⚠ 妈妈系统（最关键！）

妈妈会随机出现，分 **6 个阶段**，留给你的反应时间逐级缩短：

```
🟢 IDLE（安全）   → 自由收集光球
🟡 走廊脚步声     → 房间出现微弱红色脉冲（早期预警）
🟠 妈妈到门外     → 门缝透出黄色走廊光，反应窗口启动！
🔴 妈妈偷看       → 必须立刻放下手机！
🔴 妈妈进房检查   → 还在玩 = 游戏结束
🟢 妈妈离开       → 等门关上后再拿起手机继续
```

### 怎么判断妈妈来了？
1. 听 — 门吱呀声、心跳声
2. 看 3D 房间 — 红色脉冲灯越来越强、门缝透光
3. 看 UI — 屏幕顶部弹出红色警告条 + 屏幕中央出现"放下手机"大按钮

### 反应窗口
| 难度 | 反应时间 |
|---|---|
| 简单 | 3.5 秒 |
| 普通 | 2.8 秒 |
| 困难 | 2.2 秒 |
| 噩梦 | 1.6 秒 |
| 地狱 | 1.2 秒 |

## 🤖 巡逻机器人

每个机器人有一个**视野扇形**（黄色锥光，警觉时变红）：
- 被照到约 0.35 秒就被抓
- 在它身后或者**用家具挡住视线**完全安全
- 接触机器人本身 = 立即被抓

> 💡 **战术**：观察巡逻路径，等机器人转身瞬间穿过；卡住家具死角窝起来等机器人路过。

## ✨ 计分与连击

- 收集一个光球 = 100 分
- **连击**：2.5 秒内连续收集，combo +1，每个分数 = 100 + combo×25
- 大胆连续收集多个相邻光球能获得高分

## 💨 冲刺

- 触发后 0.18 秒内速度 ×2.3
- 冷却 1.5 秒
- **用法**：突破机器人视野扇形、拉开距离、抢救即将巡逻到的光球

## 🏆 通关条件

- 在时限内收集**所有**金色光球
- 任何形式失败：被妈妈发现 / 被机器人抓到 / 时间到

## 📈 难度建议

| 玩家 | 推荐难度 |
|---|---|
| 第一次 | 简单 |
| 想感受核心玩法 | 普通 |
| 标准挑战 | 困难 |
| 反应党 | 噩梦 |
| 自虐党 | 地狱 |

## 🎓 进阶技巧

1. **先观察后行动**：进关后不急着动，先看机器人巡逻路径与光球分布
2. **分批收集**：不要追求一次连击全收，先清离机器人远的安全光球
3. **预警保留体力**：警告响起时不要冲刺逃跑，**先停下放下手机**最稳
4. **冲刺保命**：机器人变红那 0.35 秒判定窗口内，冲刺脱离视野能逃
5. **用障碍**：机器人和你之间有家具，它就**看不见**你 — 视线被遮挡是潜行核心
6. **记住朝向**：机器人头顶有蓝/红"眼"指示朝向，永远走向它的"屁股"方向

## 🔧 运行游戏

```bash
# 安装依赖
npm install

# 本地开发服务器（手机访问：浏览器打开 http://<电脑IP>:5173）
npm run dev

# 构建生产版本
npm run build
npm run preview
```

服务器启动后会监听局域网，手机在同一 Wi-Fi 下用电脑 IP + `:5173` 即可访问。

## 🐛 注意事项

- 首次点击/触摸后浏览器才会启用音效（浏览器策略限制）
- 移动端 Safari 需要从 home screen 添加为 App 才能进入沉浸模式
- 推荐 Chrome / Edge / Safari 最新版
